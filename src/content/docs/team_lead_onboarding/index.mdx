**There are 2 gmail accounts that are used for YouDescribe development:**

#### YouDescribe SFSU Gmail Account
This Account is used for the following purposes:
- Google Cloud Platform
- Google Drive
- Google Cloud Storage
- YouDescribeSFSU AWS account

#### YouDescribe Gmail Admin Account
This Account is used for the following purposes:
- Cloudflare
- Gitlab
